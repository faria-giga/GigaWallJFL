# Giga Wall JFL

Plataforma digital independente para publicação, gestão e distribuição de conteúdos multimédia e arquivos digitais.

## Como executar

Este projeto é um PWA estático. Basta abrir o `index.html` em um servidor web.

## GitHub Actions

- **Deploy**: Automático via GitHub Pages.
- **Android APK**: Vá na aba 'Actions' e execute o workflow 'Build Android APK' para gerar seu arquivo instalável.
